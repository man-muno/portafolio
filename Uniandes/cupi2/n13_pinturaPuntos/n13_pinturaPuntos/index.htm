Descripción

  
 
   

Enunciado

Es una aplicación que va permitir la edición de dos tipos de imágenes:

Imagen dispersa por puntos:

Una imagen en blanco y negro es una estructura de dos dimensiones en la cual, en cada posición, se almacena blanco (false) o un negro (true). A cada posición también se le conoce como píxel. Se dice que una imagen es dispersa cuando el número de píxeles blancos es mucho mayor que el número de píxeles negros. El formalismo seleccionado para representar una imagen dispersa es el siguiente:

 

X(0,0)

X(0,1)

X(0,3)

…

X(0,N-1)

X(1,0)

 

 

 

 

X(2,0)

 

 

 

 

…

 

 

X(i,j)

 

X(M-1,0)

 

 

 

X(M-1,N-1)
